Weather Optimizer

Welcome to Weather Optimizer!
You can use this app to check weather forecasts or geographical data for a given location.
Full forecast - returns all data from all data sources (temperatures every 3 hours, precipitation every 3 hours, precipitation chance, sunrise and sunset hours for each day).
Quick forecast - returns shortened information (highest temperature for every day, precipitation chance, sunrise and sunset hours for each day).
Geographical data - returns latitude, longitude and the name of given location (if found).
